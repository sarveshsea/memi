A storefront-ready registry for shadcn apps. It packages product card, product badge, input, and button patterns around conversion-focused ecommerce UI.

**Palette:** natural commerce green + warm discount gold  
**Vibe:** product-led storefront  
**Components:** Button, Card, Badge, Input, ProductCard

## Install

```bash
memi add ProductCard --from @memoire-examples/ecommerce
```

## Best for

- Product grids and collection pages
- SaaS billing/plan cards
- Commerce experiments that need reusable product UI

## What you get

- Tailwind v4 `@theme` tokens (`tokens/tokens.css`)
- W3C DTCG token manifest (`tokens/tokens.json`)
- Five React components using CSS variables, not hardcoded hex
- `ProductCard` organism with sale badge, price, rating copy, and CTA
