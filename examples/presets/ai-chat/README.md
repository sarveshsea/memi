A dark, tokenized AI chat registry for shadcn apps. It gives builders the core chat primitives they need after Claude/v0 creates a first pass: message surfaces, composer input, CTA button, card, badge, and tokens.

**Palette:** near-black interface + AI mint accent  
**Vibe:** polished LLM product surface  
**Components:** Button, Card, Badge, Input, ChatMessage, ChatComposer

## Install

```bash
memi add ChatComposer --from @memoire-examples/ai-chat
```

## Best for

- AI assistants and copilots
- RAG/search interfaces
- Chat-first SaaS apps that need a stronger design system baseline

## What you get

- Tailwind v4 `@theme` tokens (`tokens/tokens.css`)
- W3C DTCG token manifest (`tokens/tokens.json`)
- Six React components using CSS variables, not hardcoded hex
- `ChatMessage` and `ChatComposer` primitives for prompt-driven products
