A conversion-focused landing page registry for code-first shadcn apps. Use it when you need a polished hero, strong CTA treatment, and the primitives that support waitlists, launches, and product pages.

**Palette:** warm editorial neutrals + launch orange accent  
**Vibe:** high-conversion marketing site  
**Components:** Button, Card, Badge, Input, HeroSection

## Install

```bash
memi add HeroSection --from @memoire-examples/landing-page
```

## Best for

- Product launches and waitlists
- SaaS marketing pages
- Founder landing pages that need one strong above-the-fold section

## What you get

- Tailwind v4 `@theme` tokens (`tokens/tokens.css`)
- W3C DTCG token manifest (`tokens/tokens.json`)
- Five React components using CSS variables, not hardcoded hex
- `HeroSection` organism that composes Button, Badge, Input, and Card primitives
