An installable auth UI registry for shadcn apps. It packages login, signup, and account-settings patterns into a tokenized system developers can add without starting in Figma.

**Palette:** trust indigo + security cyan  
**Vibe:** polished SaaS auth flow  
**Components:** Button, Card, Badge, Input, AuthCard

## Install

```bash
memi add AuthCard --from @memoire-examples/auth-flow
```

## Best for

- Login and signup pages
- Account settings entry points
- B2B products that need secure, clean auth surfaces

## What you get

- Tailwind v4 `@theme` tokens (`tokens/tokens.css`)
- W3C DTCG token manifest (`tokens/tokens.json`)
- Five React components using CSS variables, not hardcoded hex
- `AuthCard` organism with email/password inputs, CTA, helper action, and trust badge
