Codex plugin

Memoire Codex plugin

Give Codex the same design-system memory, MCP tools, Tailwind and shadcn diagnostics, Figma context, and Atomic Design workflow that Memoire already ships for local agents.

Install from the Git-backed marketplace

codex plugin marketplace add sarveshsea/m-moire --ref main --sparse .agents/plugins --sparse plugins/memoire

Then open /plugins in Codex and install Memoire from the marketplace list.

Install through npm

npm i -g @sarveshsea/memoire
memi agent install codex-plugin

What Codex gets

  • Memoire skill context for UI design, Figma, design systems, shadcn/ui, Tailwind, and Atomic Design.
  • MCP server wiring for memi mcp start --no-figma, safe for headless discovery.
  • Evidence from memi diagnose, tokens, specs, shadcn registries, and suite recipes before broad frontend edits.
